## Problem

1. **Firestore Index Errors**: Feed queries fail with 'The query requires an index' errors when filtering by environment profile (residence type, light direction, experience level).
2. **Post Creation Blank Page**: Accessing `/posts/create` shows a blank page instead of redirecting to the wizard.
3. **Syntax Errors**: Multiple wizard step pages have stray quote characters causing build failures.

## Solution

- Add composite Firestore indexes for all Feed filter combinations
- Create redirect page at `/posts/create` root
- Fix syntax errors in wizard step pages
